### usage

```
$ mkdir config
$ vim default.json
```

#### sample of default.json
```
{
    "GCMAPIKey": "Your FCM API key"
}
```

#### start server
```
$ node server.js
```
see  
`localhost: 3000`

##### webpush test from App server. (you can execute this after doing browser test)
```
$ node push.js
```
